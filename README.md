# Andrea AI - Telegram Bot

Andrea is an advanced AI agent powered by **Hermes 3** and **Ollama**, designed for direct and analytical communication.

## Architecture
- **Engine:** [Hermes Agent](https://github.com/NousResearch/Hermes-Agent)
- **Model:** Llama 3.2 3B (Running locally on VPS via Ollama)
- **Platform:** Telegram
- **Deployment:** Easypanel (Docker Compose)

## Features
- **Hands-free Automation:** Fully automated deployment pipeline.
- **Local LLM:** Privacy and speed by running models on the same VPS.
- **Hermes Soul:** Custom personality defined in `.hermes/SOUL.md`.

## Project Structure
- `docker-compose.yml`: Defines the Ollama and Hermes services.
- `.hermes/SOUL.md`: System prompt and core identity.
